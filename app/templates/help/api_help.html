Public API

Malice's Public API lets you upload and scan files, submit and scan URLs, access finished scan reports and make automatic comments on URLs and samples without the need of using the HTML website interface. In other words, it allows you to build simple scripts to access the information generated by Malice.s

File Upload

Upload a Sample

This will upload your sample to Malice where it will be scanned with AV and submitted to the sandbox for dynamic analysis

http://{{ my_url }}/api/v1/file/scan

This API call expects multipart/form-data parameters, the string part of the the call should have the following parameter:

  • apikey: your API key.

Get File Report

To retrieve the analysis report for your sample use the following.

http://{{ my_url }}/api/v1/file/report

With the Following two HTTP POST parameters:

  • resource: a md5/sha1/sha256 hash will retrieve the most recent report on a given sample.
  • apikey: your API key.

Search Intel

Search File Hash

To check Intel sources for a file use the following.

http://{{ my_url }}/api/v1/search/file

With the Following two HTTP POST parameters:

  • resource: a md5/sha1/sha256 hash will be sent to all available Intel sources.
  • apikey: your API key.
